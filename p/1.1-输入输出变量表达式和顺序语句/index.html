变量

变量用来储存信息

  • bool false = 0, true = 1

bool false/true 1byte

  • char ‘c’, ‘a’, ' ‘, ‘\n’ 字符型, 用单引号引用’ ' 1byte

  • int -2147483648，2147483647 4byte

  • float 1.23, 2.5, 1.235e2，单精度浮点数，有效数字6-7位 4byte

  • double 双精度浮点数，15-16位有效数字 8byte

  • long long 更长的整型-2^63~2^63-1 int扩展版 8byte

  • long double 18-19位有效数字 16byte

B byte b bit

1 byte = 8 bit

8Mb = 1mB

#include <iostream>
#include <cstdio>

using namespace std;

int main()
{
    int a, b;
    scanf("%d%d", &a, &b); //第一个引号里是字符串，字符串里读入两个变量，分别是整数类型，%d是指读入整数类型，再读到a和b里面去，要加&符号，不加报错
    printf("%d %d\n", a + b, a * b);//%d输出整数。第一个位置的数放到第一个%d中去，第二个位置的数放到第二个%d中去。
    printf("a+b=%d\na*b=%d\n", a+b, a*b); //格式化输出
    
    return 0;
}

scanf printf要判断变量类型

如果是浮点数把%d换成%f，保留两位小数换成%.2f，三位小数%.3f，保留几位就是点几

scanf printf的功能和cout cin类似，但是比后者强，所以常用前者，后者效率更低

字符串输入输出

#include <iostream>
#include <cstdio>

using namespace std;

int main()
{
    char a, b;
    scanf("%c%c", &a, &b);
    printf("%c %c\n", a, b);
    
    return 0;
}c

%c可以读入空格。所以对于这个程序，输入ac，输出是a c。所以要注意过滤空格。

%d不会读入空格。

cin 也不会读入空格。

double用%lf表示

long long用%lld表示

不同变量的输出方式：

int: %d

float: %f

double: %lf

char: %c

long long: %lld

long double: %llf

bool: %d

表达式

/ 除法，如果是浮点数就是相除，整数的话是整除

%取模，返回余数，被除数是负数，结果是负数。完全取决于前面的数。只有整数才能取模。

a++，++a对a的影响是把a加1

c = a++ 先把a的值赋给c然后再给a+1

d=++b 先把b+1，再把+1后的值赋值给d

–自减同理。++ 和 –只能对整数操作。

b += a, b = b +a

b -=a, b = b - a

b /=a; b = b / a;

b %= a, b = b % a;

b *= a, b = b * a

  1. 先把第k位移到最后一位：n»k;
  2. 看各位是几，x&1；
  3. n » k & 1;

lowbit(x)

树状数组的基本操作

作用是返回x的最后一位1是多少

x = 1010，lowbit(x) = 10;

x = 101000, lowbit(x) = 1000;

lowbit的实现：x & -x

-x是原数的补码（取反加1）

-x = ~x + 1;

x & -x = x & (~x + 1);

​ x = 1010…100…000;

​ ~x = 0101…011…111;

​ ~x + 1 = 0101…100…000;

x & (~x + 1) = 0000…100…000;

应用：统计x中1的个数，每次把最后一个1减掉，当x等于0的时候，里面就没有1了。

原码：x = 000…1010 （32位）

反码：x = 111…0101（每个位置都取反）

补码：x = 111…0110 （反码+1）

unsigned int 无符号整数

负数用补码来实现，计算机的低沉实现没有减法，用加法来做减法。

-x = 0 - x = 000…00 -x

vector alls; //存储所有待离散化的值

sort(alls.begin(), alls.end()); //将所有值排序

alls.erase(unique(alls.begin), alls.end()), alls.end()); //去掉重复元素 。unique将数组中所有元素去重，并返回去重后的尾端点，并把重复元素放在后面.

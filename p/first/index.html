利用hugo创建个人博客

  1. 安装chocolatey、hugo、git

    https://www.cnblogs.com/kreo/p/4685988.html

    https://gohugo.io/getting-started/installing/

    #用管理员打开PowerShell，首先改变PowerShell策略,允许运行签名脚本

    Set-ExecutionPolicy RemoteSigned

    #然后直接执行代码进行安装:

    Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

    #用Chocolatey直接安装posh-git

    choco install poshgit

    #用Chocolatey直接安装hugo

    choco install hugo -confirm

  2. 新建一个GitHub库

    新建一个 GitHub repository，库名为 username.github.io(后缀一定不能少，惨痛教训），username 即你的 GitHub 账号 username(也很重要)。

    生命仓库后，按照引导创建SSH密钥

    git config --global user.name "yourname"
git config --global user.email "youremail" 
cd ~/.ssh
ssh-keygen -t rsa -C “youremail”	#生成密钥，按三个回车，密码为空
#最后得到了两个文件：id_rsa和id_rsa.pub，复制id_rsa.pub的内容
#登录github后，进入个人设置settings--->ssh and gpg keys-->new ssh key 添加即可。title自行命名

  3. 创建Hugo网站

    # 进入想要存放网站文件夹的目录 cd /last/ # 新建一个 Hugo 网站

    hugo new site zblog

  4. 修改主题

    #在zblog中新建一个 themes 的文件夹用于存储主题。

    Hugo Themes: https://themes.gohugo.io

    cd themes
   
git clone https://github.com/g1eny0ung/hugo-theme-dream.git dream

  5. 新建文章

    #进入想要存放网站文件夹的目录 cd .\zblog

    #新建文章

    hugo new post/first.md

  6. 本地预览网站效果

    #启动hugo

    hugo server -D

  7. 构建hugo网站

    在zblog下执行hugo 命令

  8. 将网站文件夹转换为git库

    在public文件夹下执行如下命令

    #初始化git库

    git init

  9. 将git本地库关联至远程库

    在public目录下执行如下命令

    git remote add origin git@github.com:zzzqiii/zzzqiii.github.io.git

  10. 提交修改至本地库

    在public下执行以下命令

    git status

    git add .

    git commit -m “Add a new post” #必不可少

  11. 将修改推至远程库

    在public下执行以下命令

    git push -u origin master

  12. 删掉在github上更改站点发布源（必做）

    github默认转到main分支上，在仓库的settings里找到GitHub Pages，把默认的main改为master。

    参考https://docs.github.com/cn/github/working-with-github-pages/configuring-a-publishing-source-for-your-github-pages-site

    附：更改默认分支

    附：about github pages

  13. 发布新文章

    在zblog下执行下面命令新建文章

    hugo new post/paper.md

    更新后

    hugo

    #编辑完后更新GitHub上的内容

    在public目录下

    git status #查看当前修改状态

    git add . #添加所有修改过的文件

    git commit -m “Add a new post” #引号内容是本次提交的说明

    git push -u origin master

    如果不再本地添加commit，在GitHub上添加，则还需在public下

    git pull origin <your-branch>

    否则会报错：error: failed to push some refs to 'github.com:zzzqiii/zzzqiii.github.io.git'

    如果报错fatal: refusing to merge unrelated histories

    则git pull origin master --allow-unrelated-histories

  14. 修改博客背景

    hugo-theme-dream

    根据里面的configure，修改config.toml，把一些参数加上。

    这里的背景图片应该是在\zblog\themes\dream\static\img中，在这个文件夹中添加自己喜欢的背景图，然后把图片的相对路径写在配置文件中。

    博客文章的封面图片应该在\zblog\public\img中，编辑md的时候，需要插入图片的话把图片的相对路径写上。

    参考主题的作者的github

  15. 博客背面添加blog

    #自我介绍

    hugo new about/me.md

    hugo new about/index.md

    编辑md

    index.md内容为

    ---
headless: true
---

  16. 插入图片

    插入本地图片：在static文件夹下新建一个文件夹，比如R_data_science_c1, 把图片放在里面，在markdown中添加，，注意不要把/static写进去。

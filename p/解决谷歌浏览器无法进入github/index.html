解决谷歌浏览器无法进入GitHub

Mar 06, 2021

谷歌浏览器无法访问GitHub

谷歌浏览器总是进不去GitHub，但是edge可以。在网上搜了一下，发现是ip地址的原因。

解决方法如下：

  1. 访问https://github.com.ipaddress.com/www.github.com，查看ip地址。

  2. 在C:\Windows\System32\drivers\etc文件夹中找到hosts文件，添加一行。前面为刚刚查到的ip。一定要改成自己ip，之前搜到别的方法，复制别人的ip没有解决问题。

    140.82.114.4   github.com

  3. 点击host文件没有权限保留更改，右键hosts，选择属性，点击安全→编辑→Users→写入，然后确定即可。修改完hosts文件之后可以再回来把写入这个勾取消掉。

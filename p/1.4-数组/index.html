1.4 数组

May 05, 2021

程序 = 逻辑 + 数据，数组是存储数据的强而有力的手段。

定义数组

int a[10], b[10]; float c[11]; double d[12]; char e[14]; string g[14];

数组初始化

int a[3] = {0, 1, 3};
int b[] = {0, 1, 3}; //定义了一个长度为3的数组。
int c[5] = {0, 1, 1}; //定义了一个长度是5的数组，没有给出的值默认是0；
char d[] = {'a', 'b', 'c'}

int f[10] = {0}; //将数组全部初始化成0的写法。

定义在函数内部的变量是随机的，同理，数组也是随机的。

局部变量定义在栈里面，全局变量定义在堆里面。

函数外面的变量全都是零。

a[0] = 1; 下标从0开始。

旋转k次

reverse(a+起始位置， a+终止位置的下一个位置)

精度不够 |x-y| < 1e-6

定义常量 const double eps = 1e-6

高精度运算，转换为数组

pow(2, n) cmath库中

const定下了就不能修改了

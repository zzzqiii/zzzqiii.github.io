1.5 字符串

May 08, 2021

输入输出

48 - 57 分别对应 0 - 9

大写英文字母从65开始.65-90

小写英文字母从97开始。97-122

差值32

字符串就是字符数组加上结束符"\0".

字符数组：用来存储字符串的数组。

printf输出字符串：%s

空字符：\0 0

char a2[] = {‘A’, ‘b’, ‘c’}

cout « a2 + 1 « endl; 从b开始输出，第二个开始

char s[100]

scanf("%s", s) 不用加取地址符号&，因为字符串本身就是指针。

cin » s;

cin » s + 1; 从第二个开始读入

cout « s « endl;

不管是cin还是scanf，读入字符串的时候不是一行的读进来，读入遇到空格、回车或者是文件结束符未知。

fgets(s, 100, stdin); 第二个参数是最多读入多少字符，第三个参数是从哪个文件读入，一般是stdin（系统里的一个变量，已经定义好了）。这个方法可以用来读入一行。

cin.getline(s, 100); 效果和fgets一样。

string s;

getline(cin, s); 这个方法也可以读入一行，但是只能读入到字符串里，不能读到字符数组中。

string只能用cin读，scanf读不了。

输出：

printf("%s\n", s);

puts(s); 等价于上面这句话，包括换行符。

字符串常用操作

下面几个函数需要引入头文件:

#include <string.h>

中也有这个函数

(1) strlen(str)，求字符串的长度，不包含结束符。

(2) strcmp(a, b)，比较两个字符串的大小，a < b 返回-1，a == b 返回0，a > b返回1。这里的比较方式是字典序！string compare

(3) strcpy(a, b)，将字符串b复制给从a开始的字符数组。

标准库类型 string

可变长的字符序列，比字符数组更加好用。需要引入头文件：

#include

string s1;

输出：printf("%s\n", s1.c_str())

不能直接用printf

puts(s1.c_str())

读入：cin.getline(s2, 1000);

getline(cin, s1)

s1.empty()判断是否为空字符。返回布尔值

s.size()返回字符串长度，很快，不用循环一边数组。

for (char c : s) cout « c « endl; 范围遍历，遍历字符串的每一个字符。

for (char c : s) c = ‘a’;

等价于：

for (int i = 0; i < size.(); i++)

{

​ char c = str[i];

​ c = ‘a’; //不改变字符串

}

如果是for (char &c : s) c = ‘a’ 或者char &c = str[i]// 会改变字符串

while (~scanf("%c", &c))

getline会把回车过滤，fgets会把回车也读进来。

substr(i, len) 返回从i开始，长度为len的一段。

stringstream在中

把字符串初始化为类似cin的一个东西

string s;

stringstream ssin(s);

ssin » a » str » b » c;

从中读出任意需要的东西。从字符串中提取需要的东西

char s[1000];

fgets(s, 1000, stdin);

int a, b;

char str[1000];

double c;

sscanf(s, “%d%s%d%lf”, &a, str, &b, &c);

printf("%d%s%d%lf", a, str, b, c)

~a等价于a != -1

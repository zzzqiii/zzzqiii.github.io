sra下载

May 10, 2021

  1. 下载

    • conda下载：https://anaconda.org/bioconda/sra-tools

    To install this package with conda run one of the following: conda install -c bioconda sra-tools conda install -c bioconda/label/cf201901 sra-tools

  2. 使用prefetch发现报错：prefetch.2.8.0 err: path not found while resolving tree within virtual file system module。网上搜索之后选择了手动下载。

  3. 手动下载：

    • https://trace.ncbi.nlm.nih.gov/Traces/sra/sra.cgi?view=software

tar zxvf sratoolkit.2.11.0-ubuntu64.tar.gz

解压，把bin添加到环境变量。

发现还是无法使用，看了一下官网，https://github.com/ncbi/sra-tools/wiki/03.-Quick-Toolkit-Configuration

需要configure。

在bin目录下vdb-config -i

然后会出来一个图形页面，按提示说的乱点就好了，主要是设置了一个下载目录。

  1. 下载SRA

prefetch --option-file sra.txt

数据默认下载到~/ncbi/public/sra/里面

会得到一个叫SRR6784805.sra的文件。其实可以认为这个文件也是一个压缩包之类的东西。 需要解压（转换为相应的格式）。

如果是fastq

fastq-dump --split-files SRR6784805.sra

如果是sff

sff-dump --split-files SRR6784805.sra

使用fastq-dump后，就得到了双端的fastq测序数据了。

但是好像比较慢，一直是…，不确定是不是在下，最后还是选择了了迅雷orz。

  1. 迅雷下载

在ncbi上搜索

下载这个生成的csv文件，复制下载链接那一列到迅雷中去。

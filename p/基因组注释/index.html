基因组注释

  1. 基因组质量评估 （busco）

安装

conda install

下载数据库文件

下载地址https://busco-data.ezlab.org/v4/data/lineages/, 我注释的是昆虫基因组，选择的是insecta_odb10.2020-09-10.tar.gz，点击右键复制链接。

wget -c https://busco-data.ezlab.org/v4/data/lineages/insecta_odb10.2020-09-10.tar.gz
tar -xzvf insecta_odb10.2020-09-10.tar.gz

运行

busco -l /home/zuozhangqi/busco_downloads/insecta_odb10 -i Genome.final.fa -o sinc_busco -m genome

-l lineages下载的数据库文件路径

-i 基因组文件

-o output，输出文件夹名称

-m mode，运行模式

801M的基因组文件，跑了5个小时。

可视化

busco的generate_plot.py脚本可以将得到的summary文件可视化。

在conda下载的busco没找到这个脚本，在busco的gitlabhttps://gitlab.com/ezlab/busco/blob/master/scripts/generate_plot.py上的下载到了。

新建一个文件夹存放summary文件，这里的summary的文件名是short_summary.specific.insecta_odb10.sinc_busco.txt，不是在run_insecta_odb10文件夹下的short_summary.txt。

运行脚本：

python3 generate_plot.py -h #查看用法

python3 generate_plot.py -wd [WORKING_DIRECTORY]

  1. 重复序列注释(repeatmodeler & repeatmasker)
# RepeatModeler
mkdir 01_repeatModeler-denovo-repeat.lib && cd 01_repeatModeler-denovo-repeat.lib
BuildDatabase -name GDB -engine ncbi ../genome.fa &>BuildDatabase_run.log
RepeatModeler -engine ncbi -pa 28 -database GDB -LTRStruct &>RepeatModeler_run.log
cd ../


# RepeatMasker
mkdir 02_delete-denovo-lib-result 03_delete-repeatmasker-lib-result 04_delete-repeamasker-noint-result
RepeatMasker -lib 01_repeatModeler-denovo-repeat.lib/RM_*/consensi.fa.classified -pa 28 -dir 02_delete-denovo-lib-result genome.fa &>RepeatMasker_run.log1
RepeatMasker -pa 28 -dir 03_delete-repeatmasker-lib-result 02_delete-denovo-lib-result/genome.fa.masked &>RepeatMasker_run.log2
RepeatMasker -pa 28 -dir 04_delete-repeamasker-noint-result -noint 03_delete-repeatmasker-lib-result/genome.fa.masked.masked &>RepeatMasker_run.log3

得到屏蔽了重复序列的基因组文件genome.fa.masked.masked

May 04, 2021

学习循环语句只需要抓住一点——代码是顺序执行的。

while 循环

  • while 95%使用频率高

循环版的if语句

#include <iostream>

using namespace std;

int main()
{
    int i = 0;
    
    if (i < 10)
    {
        cout << i << endl;
        i ++;
    }
    
    while (i < 10)
    {
        cout << i << endl;
        i ++;
    }
    return 0;
}

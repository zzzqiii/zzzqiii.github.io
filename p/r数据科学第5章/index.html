对分布进行可视化表示

对分布进行可视化的方法取决于变量是分类变量还是连续变量。

分类变量在R中通常保存为因子或字符向量。想要检查分类变量的分布，可以使用条形图：

ggplot(data = diamonds) +
  geom_bar(mapping = aes(x = cut))

5.4缺失值

#发现异常值后，有两种做法
#1. 将带有可疑行全部丢弃（不推荐）
diamonds2 <- diamonds %>%
  filter(between(y, 3, 20))

#2. 用缺失值来代替异常值
diamonds2 <- diamonds %>%

箱线图

  • 下面的边表示分布的第25个百分位数，上面的边表示分布的第75个百分位数。
  • 箱子中间的横线表示分布的中位数。
  • 圆点是落在箱子上下两边1.5倍四分位距外的观测，这些离群点就是异常值。
  • 从箱子上下两边延伸出的直线（须）可以到达分布中最远的非离群点。

(5) 二维图形可以显示一维图形中看不到的离群点。例如，以下图形中的有些点具有异常的x 值和y 值组合，这使得这些点成为了离群点，即使这些点的x 值和y 值在单独检验时似乎是正常的。

ggplot(data = diamonds) +
  geom_point(mapping = aes(x = x, y = y)) +
  coord_cartesian(xlim = c(4, 11), ylim = c(4, 11))

coord_cartesian 默认情况，指定参数则控制图形特定区域放大显示.

coord_cartesian(xlim = NULL, ylim = NULL, expand = TRUE)

如果不加参数，就是默认的图形，和不加这个函数一样

  • xlim 和 ylim 各接一个二元数值向量，控制显示的区域范围，原理不是截取，而是放大该区域，原有图形不变，超出此范围的点也可以照常显示
  • expand 为T则默认指定xlim轴两端还留一些空隙，F则控制范围即为图形的边缘

5.6模式和模型

ggplot(data = faithful) +
  geom_point(mapping = aes(x = eruptions, y = waiting))

数据中的模式提供了关键线索。如果两个变量之间存在系统性的关系，那么这种关系就会在数据中表现为一种模式。

  • 如果两个变量是共同变化的，就可以使用一个变量的值来更好地预测另一个变量的值。
  • 如果相关变动可以归因于一种因果关系，那么就可以使用一个变量的值来控制另一个变量的值。

模型是用于从数据中抽取模式的一种工具。

#切割质量与价格之间的关系很难理解，因为切割质量和克拉数和价格之间的关系是紧密相关的。使用模型去除价格和克拉数之间的强关系，继续研究剩余的微妙关系。

library(modelr)
mod <- lm(log(price) ~ log(carat), data = diamonds)

diamonds2 <- diamonds %>%
  add_residuals(mod) %>%
  mutate(resid = exp(resid))
#拟合了一个模型，可以根据carat预测price，并计算出残差（预测值和实际值之间的差别）

ggplot(data = diamonds2) +
  geom_point(mapping = aes(x = carat, y = resid))


ggplot(data = diamonds2) +
  geom_boxplot(mapping = aes(x = cut, y = resid))

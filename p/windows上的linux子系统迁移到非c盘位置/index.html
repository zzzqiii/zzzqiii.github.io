Windows上的Linux子系统迁移到非C盘位置

Apr 02, 2021

的 Windows上安装Linux子系统非常方便，但是默认安装在C盘，眼见着C盘空间越来越小，于是想着把它移到别的盘。

使用工具：[LxRunOffline](Releases · DDoSolitary/LxRunOffline · GitHub).

我下载的是LxRunOffline-v3.5.0-msvc

解压后，通过gitbash进入到LxRunOffline-v3.5.0-msvc目录，

执行一下命令：

$ ./LxRunOffline.exe list
Ubuntu  #显示安装的Linux子系统

$ ./LxRunOffline.exe move -n Ubuntu -d /e/WSL/Ubuntu
#把上条命令显示的Ubuntu移动到别的盘

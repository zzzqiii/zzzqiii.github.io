printf语句

保留几位小数，比cout简单。

printf("%.2lf\n", f);

printf("%5d!\n", a); //最少保留5位字符，在数字左边补上空格

printf("%-5d!\n", a); //右边补上空格

printf("%05d!\n", a); //补上0

printf("%5.1lf", f)；//浮点数补位

printf("%05.1lf", f); //空位补0

printf("%-5.1lf\n", f); //右边补位

基本if-else语句

#include <cstdio>
#include <iostream>

using namespace std;

int main()
{
    int score;
    cin >> score;
    if (score >= 60)
    {
        cout << "及格";
    }
    else
    {
        cout << "不及格";
    }
}

变形：

else可以省略；

判断语句中大括号内只有一句话的时候大括号可以省略；

大括号内部也可以嵌套语句；

c++以分号来判断语句结束，和缩进与换行无关。

常用判断方式：

大于 a > b 小于 a < b 大于等于 a >= b 小于等于 a <= b 等于 a == b 不等于 a != b

else对应if是就近原则。

变量作用域：括号里面可以用外面的变量，外面不能用里面的

单引号：一个字符

双引号：字符串

读入字符时，cin会忽略空格，scanf不会

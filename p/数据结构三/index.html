哈希表

存储结构：

  1. 开放寻址法（y总常用，因为只需要开一个数组）；拉链法（都很常用）
  2. 字符串哈希方式

哈希表的作用：把一个庞大的空间或值域映射到比较小的空间，一般映射到0~N。

拉链法：

模一般要取质数，且离2的整次幂尽可能的远，冲突的概率最小。

插入：

e[idx] = x, ne[idx] = h[k], h[k] = idx ++;

维护一个集合，支持如下几种操作：

I x，插入一个数 x ； Q x，询问数 x 是否在集合中出现过； 现在要进行 N 次操作，对于每个询问操作输出对应的结果。

输入格式 第一行包含整数 N ，表示操作数量。

接下来 N 行，每行包含一个操作指令，操作指令为 I x，Q x 中的一种。

输出格式 对于每个询问指令 Q x，输出一个询问结果，如果 x 在集合中出现过，则输出 Yes，否则输出 No。

每个结果占一行。

数据范围 1≤N≤105 −109≤x≤109 输入样例： 5 I 1 I 2 I 3 Q 2 Q 5 输出样例： Yes No

#include <iostream>
#include <cstring>

using namespace std;

const int N = 100003;

int h[N];
int e[N], ne[N], idx;

void insert(int x)
{
    int k = (x % N + N) % N;
    e[idx] = x, ne[idx] = h[k], h[k] = idx ++;
}

bool find(int x)
{
    int k = (x % N + N) % N;
    for (int i = h[k]; i != -1; i = ne[i])
        if (e[i] == x)  return true;
    return false;
}

int main()
{
    int n;
    cin >> n;
    memset(h, -1, sizeof h);
    while (n --)
    {
        char op[2];
        int x;
        scanf("%s%d", op, &x);
        if (*op == 'I') insert(x);
        else
        {
            if (find(x))    puts("Yes");
            else    puts("No");
        }
    }
    return 0;
}

开放寻址法

数组开数量的两倍。

哈希值为k，先看第k个位置是否被占，如果有的话就去下一个坑位，直到找到一个空的坑位，就把x放进去（上厕所）。

添加：先找到k，从第k个坑位开始往后找，直到找到第一个空的坑位为止，把它插入。

查找：从第个坑位开始，从前往后找，每一次先看当前坑位有没有数，如果有并且是x的话，就找到了x。如果有数但不是x，就看下一个坑位，如果坑位没数，说明x不存在。

删除：和查找类似，从前往后找，如果找到了x，一般不会真的删掉，而是在数组中打一个特殊的标记，标记x有没有被删掉。

90%的情况只用到查找和添加。

#include <iostream>
#include <cstring>

using namespace std;

const int N = 200003, null = 0x3f3f3f3f;

int h[N];

int find(int x)
{
    int k = (x % N + N) % N;
    while (h[k] != null && h[k] != x)
    {
        k ++;
        if (k == N) k = 0;
    }
    return k;
}

int main()
{
    int n;
    cin >> n;
    memset(h, 0x3f, sizeof h);
    while (n --)
    {
        char op[2];
        int x;
        scanf("%s%d", op, &x);

        else
        {
            if (h[k] != null)   puts("Yes");
            else    puts("No");
        }
    }
    return 0;
}

字符串哈希方式

O(1)

用处：快速判断两个字符串是否相等

从k进制的角度把一个字符串看成数字。

kmp能做到哈希表基本都可以，除了循环节。

字符串前缀哈希法

给定一个长度为 n 的字符串，再给定 m 个询问，每个询问包含四个整数 l1,r1,l2,r2 ，请你判断 [l1,r1] 和 [l2,r2] 这两个区间所包含的字符串子串是否完全相同。

字符串中只包含大小写英文字母和数字。

输入格式 第一行包含整数 n 和 m ，表示字符串长度和询问次数。

第二行包含一个长度为 n 的字符串，字符串中只包含大小写英文字母和数字。

接下来 m 行，每行包含四个整数 l1,r1,l2,r2 ，表示一次询问所涉及的两个区间。

注意，字符串的位置从 1 开始编号。

输出格式 对于每个询问输出一个结果，如果两个字符串子串完全相同则输出 Yes，否则输出 No。

每个结果占一行。

数据范围 1≤n,m≤105 输入样例： 8 3 aabbaabb 1 3 5 7 1 3 6 8 1 2 1 2 输出样例： Yes No Yes

#include <iostream>

using namespace std;

typedef unsigned long long ULL;

const int N = 100010, P = 131;

ULL h[N], p[N];
int n, m;
char str[N];

ULL get(int l, int r)
{
    return h[r] - h[l - 1] * p[r - l + 1];
}

int main()
{
    scanf("%d%d%s", &n, &m, str + 1);
    p[0] = 1; //初始化为1
    for (int i = 1; i <= n; i ++)
    {
        p[i] = p[i - 1] * P;
        h[i] = h[i - 1] * P + str[i];
    }
    
    while (m --)
    {
        int l1, r1, l2, r2;
        scanf("%d%d%d%d", &l1, &r1, &l2, &r2);
        if (get(l1, r1) == get(l2, r2)) puts("Yes");
        else puts("No");
    }
    return 0;
}

#include <iostream>

using namespace std;

typedef unsigned long long ULL;

const int N = 100010, P = 131;

int n, m;
char str[N];
ULL h[N], p[N]; //p数组用来存储p的多少次方。h数组用来存储字符串前缀的哈希值。

ULL get(int l, int r)
{
    return h[r] - h[l - 1] * p[r - l + 1];
}

int main()
{
    scanf("%d%d%s", &n, &m, str + 1);
    
    p[0] = 1; //P的零次方为1
    for (int i = 1; i <= n; i ++)
    {
        p[i] = p[i - 1] * P;
        h[i] = h[i - 1] * P + str[i];
    }
    
    while (m --)
    {
        int l1, r1, l2, r2;
        scanf("%d%d%d%d", &l1, &r1, &l2, &r2);
        
        if (get(l1, r1) == get(l2, r2)) puts("Yes");
        else puts("No");
    }
    return 0;
}

STL

c++ reference

vector

变长数组，数组长度可以动态变化，倍增的思想。

系统为某一程序分配空间时所需的时间基本上与空间大小无关，与申请次数有关。

每次数组长度不够时，乘以2，再把之前的元素复制过来。

size() 返回元素个数

empty() 返回是否为空

clear() 清空，不是所有容器都有。

front() 返回第一个数 back() 返回最后一个数

push_back() 向vector最后插入一个数 pop_back() 删除最后一个数

begin() vector的第0个数 a[0]的迭代器 end() 最后一个数的后面一个数，即a[a.size()]

支持随即选址

支持比较运算，按字典序来比

pair<int, int>

存储一个二元组

p.first 取得第一个元素

p.second 取得第二个元素

支持比较运算，按字典序来进行比较，以first为第一关键字，second为第二关键字。

当某个东西有两个不同的属性，就可以用pair来存储。

pair也可以存储三个属性：pair<int, pair<int, int»p;

string

字符串，substr() 返回子串, c_str() 返回string对应数组的头指针。

size() 返回字符个数

empty() 判断字符串是否为空

clear() 把字符串清空

queue

push() 往队尾插入一个元素

front() 返回队头元素

back() 返回队尾元素

pop() 从队头弹出

size() / length() 返回字符串长度

empty()

没有clear()

q = queue (); 如果要清空，需要重新构造

top() 返回堆顶元素

pop() 把堆顶弹出

没有clear函数

push() 往栈顶添加一个元素

top() 返回栈顶元素

pop() 弹出栈顶元素

size()

empty()

deque

双端队列 加强版的vector 速度稍慢

队头队尾都可插入弹出，并且可以支持随机访问。

size()

begin() / end()

set, map, multiset, multimap

基于平衡二叉树（红黑树）实现，本质上是动态维护有序序列。

size() empty() clear()

begin() end() 支持++ – 操作，返回前驱（前面一个数）和后继（后面一个数），时间复杂度是O(logn)

set 中不能有重复元素，multiset中可以有重复元素

set/multiset

​ insert() 插入一个数

​ find() 查找一个数，不存在则返回end()迭代器

​ count() 返回某一个数的个数

​ erase()

​ (1)输入是一个数x，删除所有x O(k + logn) k是x的个数

​ (2)输入是一个迭代器，删除这个迭代器

​ lower_bound() 返回大于等于x的最小的数的迭代器

​ upper_bound() 返回大于x的最小的数的迭代器

map / multimap

​ insert() 插入的数是一个pair

​ erase() 输入的参数是pair或者迭代器

​ find()

​ 时间复杂度是O(logn)

unordered_set, unordered_map, unordered_multiset, unordered_multimap

基于哈希表来实现

和上面类似，增删改查的时间复杂度是O(1)

不支持lower_bound()/upper_bound()和迭代器的++ –，和所有排序有关的操作都不支持。

bitset

压位

bitset<10000> s; //定义长度为10000的bitset

支持各种位运算

~, &, |, ^

>>, «

==, !=

[ ] 取出某一位是0是1

count() 返回有多少个1

any() 判断是否至少有一个1

none() 判断是否全为0

set() 把所有位置成1

set(k, v) 将第k位变成v

reset() 把所有位变成0

flip() 把所有位取反

flip(k) 把第k位取反
